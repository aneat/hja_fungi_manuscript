This repository provides code and metadata associated with the manuscript "Host specificity of fungal functional groups covaries with climate: Implications for intraspecific plant interactions". The metadata can be found in the "metadata" folder and the code can be found in the "code" folder.

The sequences used in this study can be found in the National Center for Biotechnology Information Sequence Read Archive (accession # PRJNA1237407) and were processed using scripts 01-03. The remainder of the scripts (04-15) use the data found in the "metadata" folder.
